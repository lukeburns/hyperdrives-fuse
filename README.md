# hyperdrives-fuse

A FUSE mount for managing a directory of hyperdrives.

## Usage

```sh
npm i -g @lukeburns/hyperdrives-fuse
hyperdrives
```

This will mount an interface at ~/Hyperdrives and replicate any drives you add to it over hyperswarm.
